# Lumi Systems

Premium startup website for Lumi Systems, an early-stage education and wellbeing technology company building Lumi as its flagship product.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- React Server Components by default

## Routes

- `/`
- `/about-lumi`
- `/solutions`
- `/how-it-works`
- `/research`
- `/partners`
- `/about`
- `/contact`

## Commands

```bash
npm run dev
npm run typecheck
npm run build
```
